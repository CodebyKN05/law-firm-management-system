# law-firm-management-system
Designed a relational database with multiple tables and foreign key relationships, applying normalization for efficient data storage. Implemented CRUD operations to manage case and client records.Focused on improving workflow efficiency and reducing manual paperwork in legal environments.

##  Features
- Manage client records
- Track legal cases
- Perform CRUD operations (Create, Read, Update, Delete)
- Structured data storage using relational database design
- Efficient data retrieval using relationships
